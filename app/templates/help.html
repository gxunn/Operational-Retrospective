这里放最常用的使用说明和反馈入口，方便快速找到问题处理方式。

常用操作

先看这几个最常见的使用点。

登录使用 admin 账号和你配置的密码登录。
API 设置在系统设置里先保存 OpenAI API Key，再使用 AI 助手和热点能力。
数据保存账号、选题、拆解、报告和日志都会保存在数据库里，刷新不会丢。
导出报告在报告详情页可以导出 Markdown、PDF 和 Word。

意见反馈

如果页面有问题，优先把现象和截图发给管理员。

建议反馈内容包含：页面名称、你的操作、报错提示、截图。

如果你是管理员，也可以直接到系统设置里修改 API、SMTP 和定时参数。
